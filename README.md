Youtube : http://www.youtube.com/mkmeorg

Adafruit Libraries required.

### Wiring For ESP12E LOLIN 

| Pin On Arduino| Pin On 5110|
| ------------- |----------------|
| GND     | G |
| VCC     | 3V3 |
| SIG    | GPIO0 (D3 on LOLIN) |
